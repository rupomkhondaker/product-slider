Easily transform your woocommerce product gallery into a carousel slider with 3 different style

== Description ==

Excellent WooCommerce Product Gallery Slider plugin allows you to display a stunning product gallery on single product page with both vertical & horizontal layout.When you have a large number of product image, you can easily demonstrate all image with this plugin.

You can display all image with horizontal & vertcial layout,This plugin also has rtl feature, auto play option, zoom option etc.

<br />
=  Plugin Features =

* 3 Gallery Layout (Horizontal Bottom Thumbs, Vertical Left thumbs, Vertical Right Thumbs)
* Excellent Admin settings with full control
* Image lightbox gallery support
* Image lightbox enable/disable option
* Image Zoom Support
* Image Zoom enable/disable option
* Responsive Layout
* RTL support
* Thumbnails display options (number of thumbnail to display)
* Navigation support
* AutoPlay support
* AutoPlay speed option
* Image Auto height support
* Color options for Navigation arrow
* Preloader for light box


== Installation ==

1. Upload `excellent-woocommerce-product-gallery-slider` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
